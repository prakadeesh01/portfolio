## 🛠️ Setup Instructions

1. **Clone the repo**
   ```bash
   git clone https://github.com/prakadeesh01/portfolio.git
   cd portfolio
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Start the dev server**
   ```bash
   npm run dev
   ```
4. **Build for production**
   ```build
   npm run build
   ```
